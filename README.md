# QR Code Generator for Portfolio

## 📌 Project Overview
This project is a Python-based QR Code Generator that creates a QR code linking directly to my personal portfolio website. When the generated QR code is scanned, it redirects the user to my live portfolio hosted on GitHub Pages.

This project demonstrates the practical use of Python libraries and shows how QR codes can be used to share digital profiles easily.

---

## 🎯 Objective
To generate a scannable QR code using Python that provides quick access to my professional portfolio website.
